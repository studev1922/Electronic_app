ELECTRONIC - APPLICATION

HƯỚNG DẪN SỬ DỤNG

I. Mô tả thông tin nhánh code

VỊ TRÍ CHỨC NĂNG MÔ TẢ
TỆP TRIỂN KHAI VÀ CẤU HÌNH (/)
/README.MD Tệp thông tin Hướng dẫn cấu hình và triển khai dự án
/db_mssql_ELECTRONS.sql Tệp scripts database Tạo database và thêm dữ liệu vào database
/config.properties Tệp cấu hình source code Cấu hình email, kết nối database, folder lưu trữ hình ảnh
TÀI NGUYÊN (/assets)
/assets Lưu trữ thông tin Thông tin ảnh và chữ hiển thị trên app
/assets/data/carousel.txt Chứa văn bản Văn bản hiển thị trên màn hình chính được random trong tệp này
/assets/data/images Chứa hình ảnh Cung cấp nơi lưu trữ hình ảnh
/assets/data/images/background Chứa hình nền Hình nền hiển thị random trên màn hình chính
/assets/data/images/account Chứa hình ảnh người dùng default.png là hình mà source code sẽ không xóa đi
/assets/data/images/category Chứa hình ảnh loại hàng
/assets/data/images/product Chứa hình ảnh sản phẩm
TÀI NGUYÊN TRONG SOURCE (/src/main/resources)
khi build file jar tệp này sẽ không được truy cập từ bên ngoài
/src/main/resources/data/randomCode.properties Lưu trữ random code Khi quên mật khẩu hoặc tạo mới tài khoản, code được random, và gửi tới email, những code này được lưu vào tệp này và có hạn là 3 phút longtime = 180000L để kiểm tra xác nhận email
/src/main/resources/html Lưu trữ các tệp html Các tệp html được hiển thị trong hệ thống lấy từ folder này
/src/main/resources/icon Lưu trữ các icons Các icon hiển thị trên giao diện được lấy từ đây
/src/main/resources/jasperReport Lưu trữ các file design jasper các tệp(jrxml) thiết kế dùng để tạo report với jasper
